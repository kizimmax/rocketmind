# Структура сайта Rocketmind

> Карта страниц маркетингового сайта (`apps/site`, localhost:3001).

## Карта URL

```
/                                        Главная
│
├── /consulting                          Консалтинг и стратегии (каталог услуг)
│   ├── /ecosystem-strategy              Экосистемная стратегия
│   ├── /digital-platform                Цифровая платформа в вашем бизнесе
│   ├── /smart-analytics                 Умная аналитика для развития бизнеса
│   ├── /team-readiness                  Диагностика готовности команды к трансформации
│   ├── /strategy-sessions               Стратегические и дизайн-сессии
│   ├── /design-sprints                  Организация дизайн-спринтов
│   ├── /skolkovo                        Получение статуса резидента Сколково
│   └── /business-readiness              Диагностика готовности бизнеса к трансформации
│
├── /academy                             Онлайн-школа (каталог курсов)
│   ├── /business-design-teams           Практикум по бизнес-дизайну для команд
│   └── /business-design-quickstart      Бизнес-дизайн. Быстрый старт
│
├── /ai-products                         AI-продукты (каталог)
│   ├── /hypothesis-testing              ИИ-агент по тестированию бизнес-гипотез
│   └── /business-modeling               SaaS ИИ-сервис моделирования бизнеса (лендинг)
│
├── /about                               О Rocketmind
│
├── /cases                               Кейсы (каталог)
│   └── /[slug]                          Шаблонная страница кейса
│
├── /media                               Медиа (блог, каталог статей)
│   └── /[slug]                          Шаблонная страница статьи
│
└── /legal
    ├── /privacy-policy                  Политика обработки персональных данных
    ├── /data-consent                    Согласие на обработку персональных данных
    └── /marketing-consent               Согласие на рекламное взаимодействие
```

## Навигация

### Шапка (Header)

| Пункт меню              | Тип       | Ссылка          |
| ------------------------ | --------- | --------------- |
| Консалтинг и стратегии   | dropdown  | /consulting     |
| Онлайн-школа             | dropdown  | /academy        |
| AI-продукты              | dropdown  | /ai-products    |
| О Rocketmind             | link      | /about          |
| Медиа                    | link      | /media          |

### Футер

**Колонки навигации:**
- Консалтинг и стратегии (8 услуг)
- Онлайн-школа (2 курса)
- AI-продукты (2 продукта)
- Компания (О Rocketmind, Кейсы, Медиа)

**Подвал:** копирайт + юридические документы.

## Шаблоны страниц

### Шаблон услуги (Consulting Service)

Используется для 8 страниц консалтинга + 1 страницы AI-продукта (hypothesis-testing).

| #  | Блок                             | Роль                                         |
| -- | -------------------------------- | -------------------------------------------- |
| 1  | H1 + Hero CTA                   | Захват внимания, оффер                       |
| 2  | Социальное доказательство        | Доверие: цифры, логотипы                     |
| 3  | О продукте (ключевая ценность)   | Раскрытие концепции услуги                   |
| 4  | Для кого это решение             | Идентификация целевой аудитории              |
| 5  | Инструменты *(опционально)*      | Методики и механики                          |
| 6  | Твёрдые результаты               | Артефакты, метрики, навыки                   |
| 7  | Прозрачный процесс (этапы)       | Пошаговый план работы                        |
| 8  | Продолжительность *(опционально)*| Сроки, ритм встреч                           |
| 9  | Уникальность (Почему Rocketmind) | Отстройка от конкурентов                     |
| 10 | Эксперт *(опционально)*          | Лица продукта                                |
| 11 | Кейсы                            | Практические примеры                         |
| 12 | Отзывы                           | Прямая речь клиентов                         |
| 13 | Призыв к действию (CTA)          | Кнопка «Оставить заявку»                     |
