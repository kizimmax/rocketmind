# 07 — Детальный гайд по UX/UI

> Статус: **черновик** | Источник: PRD MVP 1.1 + @rocketmind/ui

---

## User Flow (основной)

```
[Лендинг] → CTA «Попробовать»
    │
    ▼
[Auth: Email] → ввод email
    │
    ▼
[Auth: Code] → ввод кода из письма
    │
    ▼
[Главный интерфейс] ← ─ ─ ─ ─ ─ ─ ┐
    │                                │
    ├── Сайдбар: выбор кейса ────────┤
    ├── Сайдбар: новый кейс ─────────┤
    ├── Чат с агентом                │
    └── Каталог агентов ─────────────┘
```

### Автобутстрап flow
```
URL /a/{agent_slug}
    │
    ├── [Не авторизован] → Auth flow → auto-создание кейса → чат
    └── [Авторизован] → auto-создание кейса → чат с агентом
```

### 4B: Область чата

#### Состояния

| Состояние | Описание |
|---|---|
| **No case** | Не выбран кейс — приветственный экран «Выберите или создайте кейс» |
| **No agent** | Кейс выбран, агент не выбран — «Добавьте агента для начала» |
| **Empty chat** | Агент выбран, нет сообщений — приветствие агента |
| **Active chat** | Список сообщений + поле ввода |
| **Agent typing** | Индикатор «агент печатает...» |
| **Error** | Ошибка отправки — «Не удалось отправить. Повторить?» |

#### Сообщения

Три типа по `role`:

| Role | Визуал | Содержимое |
|---|---|---|
| **user** | Справа, акцентный фон | Текст пользователя |
| **assistant** | Слева, нейтральный фон | Markdown-текст, рендерится |
| **system** | По центру, выделенный стиль | Уведомления, CTA-кнопки (оплата, summary) |

#### Системные сообщения с CTA

n8n может отправлять системные сообщения с кнопками:

```json
{
  "role": "system",
  "content": "Ваш отчёт готов. Для получения полной версии оформите подписку.",
  "metadata": {
    "cta": {
      "label": "Оформить подписку",
      "url": "https://pay.example.com/checkout/123"
    }
  }
}
```

| Элемент | Компонент | Примечание |
|---|---|---|
| CTA-кнопка | `<Button>` | @rocketmind/ui, opens external link |
| Текст сообщения | Markdown → HTML | Нужен markdown renderer |

#### Поле ввода

| Элемент | Компонент | Примечание |
|---|---|---|
| Textarea | `<Textarea>` | @rocketmind/ui, auto-resize |
| Кнопка «Отправить» | `<Button>` | @rocketmind/ui, icon-only |

### Компоненты

| Элемент | Компонент | Примечание |
|---|---|---|

| Сообщение | Кастомный ChatMessage | Нужно создать |

## Edge Cases

| Сценарий | Поведение |
|---|---|
| Автобутстрап с несуществующим slug | Показать ошибку «Агент не найден» с кнопкой «К каталогу» |
| Потеря соединения во время чата | Banner «Нет подключения», retry при восстановлении |
| Очень длинное сообщение от агента | Markdown рендеринг с code blocks, scroll |
| Пользователь отправляет пустое сообщение | Кнопка «Отправить» disabled при пустом textarea |
| Код авторизации истёк | Показать ошибку + кнопку «Отправить новый код» |
| Сессия истекла | Редирект на Auth с toast «Сессия истекла» |
| Максимум кейсов | Если есть лимит — показать «Архивируйте кейс для создания нового» |
